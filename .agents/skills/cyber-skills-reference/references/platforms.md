# Cybersecurity Training Platforms

Full reference list, grouped by category, sourced from
[joe-shenouda/awesome-cyber-skills](https://github.com/joe-shenouda/awesome-cyber-skills)
(MIT License, see `../LICENSE.txt`). All entries are legal practice
environments - vulnerable-by-design apps, sandboxed wargames, or free course
material. Use only against systems you own, a local VM/container you
control, or the platform's own hosted lab environment.
